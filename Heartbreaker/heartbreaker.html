About the Game

Heartbreaker is a first-person 3D platformer made in Unreal Engine 5, developed by Disco Angels — a six-person student team of three programmers, one artist, one composer, and voice actors. The game is set in the world of Elcoro, where the player must literally crash the hardened hearts of its inhabitants to restore joy to the world.

The game won first place in the Best Gameplay category at Game Access Conference 2024 in Brno, Czech Republic, and first place in Game Design at ZTGK 2023, along with special prizes from Techland and other industry partners. The code was hosted on an AWS server using Perforce Helix Core and is not public as of now.

My Contributions

I was one of three programmers on the project and touched a wide range of systems — from core mechanics and AI to UI, visual effects, and infrastructure.

  • Designed and implemented core player movement mechanics and the checkpoint system.
  • Built enemy AI behaviours.
  • Authored a dissolve shader for exploding enemies + set up Chaos Geometry Collections.
  • Implemented a music blending system that transitions dynamically between tracks.
  • Authored a custom subtitles system with support for image insertion and contributed to the UI system (ex. death screen effect).
  • Created particle effects and other visual polish.
  • Contributed to both gameplay design and level design alongside the team.
  • Set up and administered the Perforce Helix Core server hosted on AWS.
