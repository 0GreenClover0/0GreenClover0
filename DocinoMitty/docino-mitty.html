About

Docino Mitty is a puzzler built around placing dominoes to form a growing kingdom. Each domino has two tiles, each displaying a value and a terrain type — forest, road, house, lake, and more. Rather than matching values as in traditional dominoes, you match terrain types: adjacent tiles of the same type form connections that compound their scores. Chain the right sequence together and the value multiplies; place carelessly and the whole thing can collapse.

The trading mechanic adds a second layer: once a run of tiles has accumulated enough value, you can cash it in for a set of cheaper, but greater in numbers, individual tiles — an "equivalent exchange" — then use those to keep building. It's a loop of building, scoring, spending, and expanding that rewards forward planning under tight constraints.

"Chain the right tiles together to create exponential combos, then use them in an equivalent exchange — one high value domino for a handful of cheaper ones."

Gameplay
  • Domino placement on a town grid
  • Terrain-type matching (not value matching)
  • Forest, road, house, lake, and more types
  • Each type has unique combo rules
  • Exponential value chaining
  • Equivalent exchange trading mechanic
  • High score pursuit loop
